Exemple d'intégration d'un iframe :

Comment intégrer une vidéo youtube ? :

  1. Aller sur youtube
  2. Choisir une vidéo
  3. Cliquer sur partager
  4. Cliquer sur intégrer
  5. Copier le code Iframe et l'importer dans votre HTML

Comment intégrer une carte Maps ? :

  1. Aller sur maps
  2. Aller sur le menu
  3. Cliquer sur partager ou intégrer la carte
  4. Copier le code Iframe et l'importer dans votre HTML
